# KD Дерево
## Теория
### Определения
- KD Дерево - особая структура данных специально создания для хранения точек в $k$ мерном пространстве. Данное Дерево является логическим потомком _бинарного дерева_ в котором, каждый "слой" делит пространство с точками на две части по разным осям координат.
- Особая структура позволяет с большой скоростью находить ближайшие точки и кластеризовать пространство

### Структура 
- Чаще всего общая структура дерева/узлов состоит из:
  1. Информация о точке
  2. Ось деления 
  3. Указатели на следующие точки с другой осью деления (левую и правую часть)
  4. (В некоторых источниках) Указатель на _родителя_ узла (см.ниже)
